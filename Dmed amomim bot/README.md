# DMED anonymous feedback Telegram bot

Бот реализует ТЗ из `ТЗ_Telegram_Bot_DMED.docx`: вход по уникальной ссылке учреждения, анонимные отзывы по сотрудникам и качеству внедрения, админ-команды, антиспам 7 дней, уведомления, экспорт Excel/PDF и еженедельные дайджесты.

## Быстрый запуск

1. Скопируйте `.env.example` в `.env`.
2. Заполните `BOT_TOKEN`, `BOT_USERNAME`, `ADMIN_PASSWORD`.
3. Запустите сервисы:

```bash
docker compose up --build
```

Миграции применяются автоматически при старте контейнера `bot`.

## Основные команды

Пользователь:

- `/start TOKEN` - вход по ссылке учреждения.
- `/feedback` - начать отзыв.
- `/my_reviews` - история своих отзывов.
- `/settings` - выбор языка.
- `/help` - краткая инструкция.

Администратор:

- `/admin` - вход по паролю.
- `/add_institution` - создать учреждение и ссылку.
- `/add_employee` - добавить внедренца.
- `/add_admin` - добавить администратора по Telegram ID.
- `/institutions` - список учреждений и управление ссылками.
- `/employees` - список сотрудников и архивирование.
- `/reviews` - последние отзывы.
- `/stats` - сводная статистика.
- `/export` - экспорт отзывов в Excel или PDF.

## Примечания

- Без deep-link токена новый пользователь не получает меню и не может оставить отзыв.
- Telegram-профиль отправителя показывается только администраторам.
- Ограничение частоты отзывов скрыто: при повторной попытке бот отвечает нейтральным сообщением.
- Основная БД - PostgreSQL. FSM-состояния хранятся в Redis, при отсутствии `REDIS_URL` используется память процесса.
